# Reasy - 読書記録管理アプリ  
公開ページ：https://doiyuki-7061.github.io/reasy/  

## 使用技術  
・React(UI構築、状態管理)  
・JavaScript/HTML/CSS  
・Node.js(開発環境)  
・Google Books API(書影・著者名の取得)  

### 開発コンセプト
読書がもっと楽しくなるように、自分の感想をいつでも振り返ることができるアプリを作りたいと思い、Reasyを開発しました。  
読んだ本の記録を残すことで、過去の自分の考えや気持ちに触れられるような、”自分だけの本棚”を目指しています。  
  
### 機能概要
・書籍のタイトル・感想・評価を入力して保存  
・保存時にGoogle Books APIから書影と著者名を自動取得  
・「本棚を開く」ボタンを押すと、記録した読書履歴をリスト表示  
・ローカルストレージに保存されるため、再読み込み後もデータ保持  
・記録後の編集・削除機能あり  
  
### 工夫した点  
・useStateとuseEffectを活用した状態管理とローカル保存  
・書影の表示で視覚的にもわかりやすいUIを実装  
・入力フォーム、本棚リスト、評価機能のコンポーネント化によるコードの整理
